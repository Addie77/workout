# GEMINI.md

## Project Overview

This is a simple iOS application named "workout" built with SwiftUI. The application is a "Hello, world!" style app that displays a globe icon and text. It is configured to run on both iPhone and iPad.

The main application entry point is `workout/workoutApp.swift`, and the main user interface is defined in `workout/ContentView.swift`.

## Building and Running

To build and run this project, you will need a Mac with Xcode installed.

1.  Open the `workout.xcodeproj` file in Xcode.
2.  Select a target device or simulator from the toolbar.
3.  Click the "Run" button (or press `Cmd+R`) to build and run the application.

## Development Conventions

The project follows the standard conventions for a SwiftUI application generated by Xcode.

*   **Language:** Swift
*   **User Interface:** SwiftUI
*   **Project Structure:** The project is organized into a main `workout` group containing the source files and an `Assets.xcassets` catalog for images and colors.
