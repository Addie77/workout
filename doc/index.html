[8] 流程: 動作教學 (彈窗)

深蹲 (Squats)

主要肌群：股四頭肌、臀大肌

1. 雙腳與肩同寬，腳尖微朝外。

2. 抬頭挺胸，核心收緊，背部打直。

3. 臀部像坐椅子一樣向後推，同時下蹲，直到大腿與地面平行。

4. 過程中保持膝蓋與腳尖方向一致。

5. 用臀腿力量發力，站回起始位置。
